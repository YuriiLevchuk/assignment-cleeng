# Assignment Cleeng

This project is a React single-page application built with Vite. It was created as part of the assignment for Cleeng front-end intership.  

## How to launch?

### 1. Clone the repository  
```bash
git clone https://github.com/YuriiLevchuk/assignment-cleeng.git
cd assignment-cleeng/project
```

### 2. Clone the repository  
```bash
npm install
```
(or you may use yarn install or pnpm install depending on your preference.)

### 3. Run in development mode
```bash
npm run dev
```
This will start the development server (commonly at http://localhost:5173/, unless configured otherwise).
